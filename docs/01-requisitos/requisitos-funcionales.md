# Requisitos Funcionales

## RF-001

El sistema deberá permitir el registro de usuarios.

## RF-002

El sistema deberá permitir el inicio de sesión mediante correo electrónico y contraseña.

## RF-003

El sistema deberá permitir la recuperación de contraseña.

## RF-004

El sistema deberá registrar lecturas de temperatura provenientes de sensores IoT.

## RF-005

El sistema deberá registrar lecturas de humedad provenientes de sensores IoT.

## RF-006

El sistema deberá almacenar cada medición en la base de datos.

## RF-007

El sistema deberá mostrar en tiempo real los datos capturados por los sensores.

## RF-008

El sistema deberá generar alertas cuando la temperatura supere los límites establecidos.

## RF-009

El sistema deberá generar alertas cuando la humedad se encuentre fuera de los rangos definidos.

## RF-010

El sistema deberá permitir consultar el historial de mediciones.

## RF-011

El sistema deberá filtrar información por rango de fechas.

## RF-012

El sistema deberá generar reportes de monitoreo.

## RF-013

El sistema deberá exportar reportes en formato PDF.

## RF-014

El sistema deberá permitir la administración de usuarios.

## RF-015

El sistema deberá registrar eventos y notificaciones generadas por el sistema.

## RF-016

El sistema deberá permitir visualizar estadísticas del proceso de compostaje.

## RF-017

El sistema deberá permitir consultar el estado actual de cada dispositivo IoT.

## RF-018

El sistema deberá permitir registrar nuevos dispositivos IoT.

## RF-019

El sistema deberá permitir activar o desactivar dispositivos registrados.

## RF-020

El sistema deberá mostrar un panel de control con indicadores clave del sistema.
