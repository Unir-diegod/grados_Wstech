# Requisitos No Funcionales

## RNF-001

La plataforma deberá estar disponible las 24 horas del día.

## RNF-002

La aplicación deberá responder consultas en menos de 3 segundos.

## RNF-003

La comunicación entre cliente y servidor deberá utilizar HTTPS.

## RNF-004

Las contraseñas deberán almacenarse utilizando algoritmos de cifrado seguros.

## RNF-005

La aplicación deberá ser compatible con navegadores modernos.

## RNF-006

La solución deberá funcionar como PWA.

## RNF-007

La interfaz deberá ser responsive.

## RNF-008

La base de datos deberá garantizar integridad referencial.

## RNF-009

El sistema deberá permitir escalabilidad para nuevos sensores.

## RNF-010

La aplicación deberá registrar eventos críticos en logs.

## RNF-011

La arquitectura deberá permitir mantenimiento modular.

## RNF-012

La información deberá respaldarse periódicamente.

## RNF-013

La plataforma deberá cumplir principios básicos de seguridad informática.
